# [dddware.github.io](http://dddware.github.io)

## Install

```
npm install
```

## GitHub auth

Create a `lib/auth.js` file as follows:

```javascript
module.exports = {
  user: 'username',
  pass: 'password'
};
```

You will then be able to perform more than 60 requests per hour!

## Build

```
npm start
```

## Publish

```
git commit && git push
```
